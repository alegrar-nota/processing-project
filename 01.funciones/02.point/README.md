# Función point - dibujar un punto

```java
// funcion point - dibujar punto

//point(x,y) en 2D
```

![1.Funcion point](imgs/01.funcion_point.png)

## Ejemplos

### 1. El punto en el medio de la pantalla
```java

void setup(){
  size(400,400);
  point(200,200); // el punto está en el medio
}
```
